# 이클립스 단축키

| 단축키            | 기능                       |
| ----------------- | -------------------------- |
| ctrl + shift + L  | 단축기 전체 목록이 나온다. |
| ctrl + +,-        | 폰트 사이즈                |
| ctrl + D          | 한 줄 삭제                 |
| ctrl + alt + down | 행 단위 복사               |
| alt + shift + A   | 멀티컬럼 편집              |
| alt + up, down    | 행단위 이동                |
| ctrl + i          | 자동으로 들여쓰기 맞춰줌   |
| ctrl + /          | 주석                       |
| ctrl + space      | 자동 완성                  |
